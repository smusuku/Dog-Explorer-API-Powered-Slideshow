# Infinite Dog App

This project is an implementation of the tutorial **"Dogs, JavaScript & An API üê∂ Fetch, Promises & Async Await"** by **Brad Schiff**.  
It uses the public Dog API to dynamically fetch dog breeds and create an animated slideshow of images.

## Features
- Fetches dog breeds dynamically from the Dog API
- Builds breed dropdown using JavaScript
- Displays animated slideshow of breed images
- Handles edge cases (1 or 2 images)
- Deployed as an Azure Website

## Technologies Used
- HTML5
- CSS3
- JavaScript (Fetch API, Promises, Async/Await)
- Azure Static Web Apps
